# GoodTunes 
A personal song journal inspired by GoodReads, but for music.
Save songs you love or want to listen to, all in one place.

## Status
Actively continuing development as of April 2026 after an initial build in 2025.

## Current Features
- User registration and login
- Personal song collection — add and remove songs
- Persistent, user-specific storage

## Roadmap
- Song reviews and ratings
- Read and comment on other users' reviews
- Follow friends and see their collections

## Tech Stack
React, React Router, Firebase Auth, Firebase Realtime Database, JavaScript, CSS

## Running Locally
1. Clone the repo
2. `npm install`
3. Add your Firebase config to `firebase.js`
4. `npm start`
